## 省流

- 当使用`sudo`时，**确保你知道自己在做什么，并知道如何撤回（尤其当你的指令来源于任何AI工具时）**。
- **禁止远程关机。**
- **禁止使用系统python。**

### 1. 系统
- 当使用`sudo`时，**确保你知道自己在做什么，并知道如何撤回（尤其当你的指令来源于任何AI工具时）**。
- 修改任何原有的系统文件时，请保存备份。
- 修改非个人文件夹下的文件时（例如`~/.bashrc`），活用注释功能保存好修改日志。

### 2. 电源
- 请尽量不要关机，关机前先查看是否有进程在运行。
- **禁止远程关机。**

### 3. 存储
- 请将个人文件保存在主目录下的个人文件夹中（例如`~/liaoyang`）。
- 如果文件被用于本机的公共基础设施建设，请保存到主目录下的公共文件夹中（`~/public`）。
- **主目录下的其他文件夹会随时、无预告清理（包括`~/下载`），请务必妥善保存文件，不要随地扔垃圾。**
- 由于本机只有一块硬盘，无法保证系统和文件的物理隔离。修复系统时无法保证文件安全，请自行备份。
- 使用git请参考[不同的项目使用不同的 Git 配置](https://yuanchaofa.com/post/git-config-path-seperation.html)。

### 4. 网络和远程
- 本机装有todesk和向日葵，请妥善使用。
- 使用校园网可以远程ssh连接，为防止ip变化导致断开可以使用[tmux](https://zhuanlan.zhihu.com/p/415515508)。
- 本机不提供网络代理。在`~/public/software`中有[v2rayN](https://v2rayn.bid/tutorial.html)，请使用自己的订阅信息，自行取用。

### 5. 软件
- **禁止使用系统python**，不要使用conda的base环境。
- 本机装有anaconda3和cuda-12.9。理论上cuda可以向后兼容，如果需要使用特定版本的cuda，请自行安装后更改自己的conda环境配置，不要修改`.bashrc`。
